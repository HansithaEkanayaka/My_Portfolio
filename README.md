# Hansitha Ekanayaka's Portfolio
 
  A modern, responsive personal portfolio website built with **React + Vite**, featuring smooth animations, an interactive particle canvas, typing effects, and a clean dark aesthetic.
 
## ✨ Features
 
- **Animated Hero Section** — Typing effect cycling through roles: Software Engineer, Full-Stack Developer, Problem Solver & more
- **Interactive Particle Canvas** — Dynamic background particles that respond to user interaction
- **Custom Cursor** — Stylish cursor with smooth tracking animation
- **Scroll Progress Bar** — Visual reading progress indicator at the top
- **Page Loader** — Elegant loading screen on initial visit
- **Smooth Reveal Animations** — Sections animate into view on scroll using IntersectionObserver
- **Skills Showcase** — Categorised tech stack with brand-coloured icons
- **Projects Grid** — Featured projects with tech stack badges and GitHub links
- **Contact Form** — Direct message form with social links sidebar
- **Responsive Design** — Fully optimised across all screen sizes
- **Back to Top Button** — Smooth scroll back to hero

## 🚀 Getting Started
 
### Prerequisites
 
- [Node.js](https://nodejs.org/) v18 or higher
- npm or yarn
### Installation
 
```bash
# 1. Clone the repository
git clone https://github.com/HansithaEkanayaka/hansitha-portfolio.git
 
# 2. Navigate into the project
cd hansitha-portfolio
 
# 3. Install dependencies
npm install
 
# 4. Start the development server
npm run dev
```
 
The app will run at `http://localhost:5173`
 
### Build for Production
 
```bash
npm run build
```
 
The optimised output will be in the `dist/` folder, ready to deploy.
 
### Preview Production Build
 
```bash
npm run preview
```
